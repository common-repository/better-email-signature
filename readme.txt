== Description ==
Better Email Signature plugin allows you add some signatures and let you choose signature that will be append to every outgoing email that WordPress sends.
refer the idea of plugin "Add Email signature" (of author David Anderson : http://goo.gl/2qqpTY) we made a new plugin which have revolution fetures.

== Installation ==

Standard WordPress plugin installation:

1. Extract the package, upload better-email-signature/ into wp-content/plugins/ (or use the Wordpress's built-in installers)
2. Activate the plugin in the 'Plugins' page.
3. Choose 'Settings' from the 'Better Email Signature' in the plugin list.
4. Follow the instructions to manage.
